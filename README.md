# ESP8266 HTTP PIR Sensor

ESP8266 HTTP PIR Sensor controlling light source via Shelly 1 Wi-Fi switch in local network. 

## Hardware example
* ESP8266-12E ch340
* Luxorparts PIR sensor (5v)
* Shelly 1
